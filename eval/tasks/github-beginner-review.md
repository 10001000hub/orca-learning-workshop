# Task: GitHub Beginner QA Loop（GH-001）

## 目的

GH-001一式（lesson / quiz / workshop）を初心者視点で通しレビューし、`review/github/GH-001/beginner-review.md`を閾値85まで改善する。

## 入力

- `curriculum/github/GH-001/lesson.md`
- `curriculum/github/GH-001/quiz.json`
- `curriculum/github/GH-001/workshop.json`
- `review/github/GH-001/beginner-review.md`
- 評価基準: `eval/criteria/beginner-review.criteria.md`

## 生成・改善の対象

- PC・IT・GitHub初心者の視点で、迷いやすさ・説明不足・用語の難しさ・画面差分への耐性・ヒント品質を評価する
- 指摘事項を`beginner-review.md`に記録する
- 指摘事項をlesson/quiz/workshopへフィードバックする（実際の修正は各Loopの担当ファイルで行う）

## 禁止事項

- レビュー対象を実機確認済みであるかのように扱うこと（GH-001は`verification.md`の通り未実施）
- 曖昧な成功判定を「問題なし」と判定すること

## 出力

更新された`review/github/GH-001/beginner-review.md`と、必要に応じたlesson/quiz/workshopへの修正指摘。
