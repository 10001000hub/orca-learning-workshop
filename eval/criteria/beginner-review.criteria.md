# Beginner Review Criteria

対象: `review/<theme>/<ID>/beginner-review.md`（GH-001の`lesson.md` / `quiz.json` / `workshop.json`一式を初心者視点で通し確認した結果）

Threshold: 85（[EVAL_LOOP_GUIDE.md](../../docs/EVAL_LOOP_GUIDE.md)参照）

## 評価観点

### 迷いやすさ

教材を進める中で、次に何をすればよいか迷う箇所がないか。

### 説明不足

初心者が疑問を持ちそうな箇所に説明が足りていないか。

### 用語の難しさ

説明なしで専門用語が使われている箇所がないか。

### 画面差分への耐性

実際の画面が多少変わっていても（レイアウト変更など）、教材の説明が破綻しないか。過度に画面依存の説明になっていないか。

### ヒント品質

workshopの`hint`が、つまずいた初心者を実際に助けられる内容か。

## Veto条件（該当したら即不合格）

- 初心者が次に何をすればいいか分からない
- Workshopの成功判定が曖昧
