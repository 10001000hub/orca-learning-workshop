# ORCA-005: Web作品をURLで公開する

## 公開とは

GitHub Pagesを使うと、リポジトリにあるWebページをURLで表示できます。URLを知っている人だけに限定されるとは限らず、インターネットへ公開されるものとして扱います。

## 公開前の最終確認

index.htmlと一緒に公開されるファイルを確認します。氏名、住所、メールアドレス、パスワード、トークン、公開したくない写真や文章があれば、公開前に取り除きます。

## 公開元

GitHub Pagesには、どのブランチのどのフォルダーを公開するかを指定します。単純な練習ページでは、index.htmlがあるブランチのルートを公開元にできます。

## 公開後の確認

設定後すぐに表示されないことがあります。公開処理の状態を確認し、案内されたURLを開きます。見出しと説明文が表示されれば、最初の成果物公開は完了です。

## まとめ：完成

OrcaでCodexへ依頼し、Webページを作り、Gitで履歴を残し、GitHubへ送り、GitHub Pagesで公開できました。公開後も、変更、コミット、pushを行うことで作品を改善できます。
