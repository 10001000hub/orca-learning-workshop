# ORCA-005 source pack

確認日: 2026-07-10

- GitHub Pagesは、指定ブランチのルートまたは`/docs`を公開元にできる。簡単なサイトではブランチからの公開が推奨される。出典: https://docs.github.com/en/pages/getting-started-with-github-pages/configuring-a-publishing-source-for-your-github-pages-site
- 公開手順はリポジトリのSettings、Pages、Build and deploymentで公開元を選び保存する。出典: 同上
- Pagesサイトはインターネットへ公開されるため、機密情報を公開元へ含めてはいけない。反映には最大10分ほどかかる場合がある。出典: 同上、および https://docs.github.com/en/pages/setting-up-a-github-pages-site-with-jekyll/about-jekyll-build-errors-for-github-pages-sites
