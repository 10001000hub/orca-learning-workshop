# ORCA-001 source pack

確認日: 2026-07-10

## 公式資料から確認した内容

- OpenAIはCodexを、コードの作成・理解・レビュー・デバッグを行えるコーディングエージェントとして説明している。
- OpenAIのCodexユースケースには、アイデアから試作品を作ること、Webサイトを作成・更新して公開URLを得ること、GitHubのプルリクエストをレビューすることが含まれる。

参照: [OpenAI Codex use cases](https://developers.openai.com/codex/use-cases)

## この環境で確認した内容

- Windows上のOrcaで、ユーザーとCodexが同じスレッド内で会話できる。
- Codexはユーザーが選択したワークスペース内のファイルを調査・作成・変更し、テストを実行できる。
- 作業中の更新と完了時の結果がOrcaの会話画面に表示される。

この3点は、ORCA-001作成時の実セッションで確認した。画面上の固有ボタン名や位置は、スクリーンショットによる確認が完了するまで教材では断定しない。
